Dans le fichier Code_Rocky.ino se retrouve le code du robot de l'équipe 11 nommée Rocky.

Ce code utilise le IDE d'Arduino et voici le lien de téléchargement :
    https://www.arduino.cc/en/software
    CHOISIR : Arduino IDE 2.0

Pour bien utiliser le code, il faut l'installation de plusieurs librairies sur le IDE.

1. Servo by Michael Margolis, Arduino
2. Adafruit Motor Sheild V2 Library by Adafruit
3. Dabble by STEMpedia
